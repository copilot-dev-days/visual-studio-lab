# LAB300 - Hands-on con GitHub Copilot en Visual Studio 2026

[![Workshop Site](https://img.shields.io/badge/Workshop-Site-blueviolet?style=for-the-badge)](https://dotnet-presentations.github.io/visual-studio-github-copilot-lab/)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue?style=for-the-badge&logo=github)](https://dotnet-presentations.github.io/visual-studio-github-copilot-lab/)

> **🚀 [Iniciar el Taller](https://dotnet-presentations.github.io/visual-studio-github-copilot-lab/)** - Guía interactiva temática del taller con todos los pasos y documentación.

Este lab te guiará en el uso de las diversas funcionalidades de GitHub Copilot en Visual Studio 2026. Comenzarás con una aplicación TinyShop parcialmente completada y usarás GitHub Copilot para completar funcionalidades faltantes y mejorar la aplicación.

## Requisitos Previos

- Visual Studio 2026 con la extensión GitHub Copilot instalada
- .NET 10 SDK
- Cuenta de GitHub con suscripción a Copilot (incluido Free)

## Descripción General del Lab

La aplicación TinyShop consta de dos proyectos principales:
- Una API backend construida con .NET Minimal APIs
- Una aplicación frontend Blazor Server

Usarás las diversas funcionalidades de GitHub Copilot para mejorar y completar esta aplicación.
