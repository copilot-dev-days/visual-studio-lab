# LAB300 - Hands-on com GitHub Copilot no Visual Studio 2026

[![Workshop Site](https://img.shields.io/badge/Workshop-Site-blueviolet?style=for-the-badge)](https://dotnet-presentations.github.io/visual-studio-github-copilot-lab/)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue?style=for-the-badge&logo=github)](https://dotnet-presentations.github.io/visual-studio-github-copilot-lab/)

> **🚀 [Iniciar o Workshop](https://dotnet-presentations.github.io/visual-studio-github-copilot-lab/)** - Guia interativo temático do workshop com todos os passos e documentação.

Este lab guiará você no uso dos diversos recursos do GitHub Copilot no Visual Studio 2026. Você começará com uma aplicação TinyShop parcialmente concluída e usará o GitHub Copilot para completar funcionalidades faltantes e aprimorar a aplicação.

## Pré-requisitos

- Visual Studio 2026 com a extensão GitHub Copilot instalada
- .NET 10 SDK
- Conta GitHub com assinatura do Copilot (incluindo Free)

## Visão Geral do Lab

A aplicação TinyShop consiste em dois projetos principais:
- Uma API backend construída com .NET Minimal APIs
- Uma aplicação frontend Blazor Server

Você usará os diversos recursos do GitHub Copilot para aprimorar e completar esta aplicação.
