# Parte 00: Explorando el código fuente con GitHub Copilot Chat

GitHub Copilot Chat te permite hacer preguntas sobre tu código y obtener respuestas inteligentes.

1. [] Abre la solución en Visual Studio 2026 si no está abierta.
1. Haz clic en GitHub Copilot Chat en la esquina superior derecha de Visual Studio y selecciona **Open Chat Window** o presiona `Ctrl+\+C` si el chat de Copilot no está abierto.
   ![Open chat window dialog](./images/1-open-copilot-chat.png)
1. [] Asegúrate de estar en modo **Ask** haciendo clic en la pestaña **Ask** en la parte inferior de la ventana de chat.
   ![Ask mode tab](./images/0-ask-mode.png)
1. [] Intenta hacer preguntas sobre la estructura del proyecto:
   - `What projects are in this solution and how do things work together?`
   - `How does the Products API work?`
1. [] Observa cómo Copilot analiza tu código fuente para proporcionar respuestas contextuales.

**Punto clave**: GitHub Copilot Chat te ayuda a comprender códigos fuente desconocidos respondiendo preguntas sobre la estructura del proyecto, la arquitectura y los detalles de implementación.
